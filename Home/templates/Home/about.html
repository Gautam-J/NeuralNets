About

A website created using the Django Web Framework, to allow users to play with machine learning models, and more specifically Neural Networks, mostly trained using the tf.keras API.

Tensorflow

Google's tensorflow was used mainly for training a lot of the models hosted on this website, more specifically, tf.keras API was used to train the neural networks.

Scikit Learn

Python's sklearn library was used for training popular machine algorithms such as Support Vector Machines and Random Forests. A lot of preprocessing was done using the same.

Django

Django Web Framework provided an organized and dynamic way to build this website, on top of python.
